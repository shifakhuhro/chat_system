PROJECT DESCRIPTION:
This project is a Chat System Database that allows users to send messages, join chat rooms, and manage attachments.

TOOLS USED:
- MySQL
- Python (Streamlit)

DATABASE FILE:
ChatSystem.sql

HOW TO RUN:
1. Import ChatSystem.sql into MySQL
2. Run app.py using Streamlit
3. Open browser at localhost:8501

FEATURES:
- User management
- Chat system
- Group chat rooms
- Message status (Sent, Read, Delivered)
- Attachments handling
- Dashboard statistics
